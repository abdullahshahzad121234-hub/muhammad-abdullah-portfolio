Tremco Construction

A fast, modern construction website built with Webflow, Lenis smooth scroll, and GSAP animations. Delivering a premium experience for industrial clients.

Tremco Construction - Hero Showcase

Project Overview

Tremco Construction is an industrial construction company that needed a modern digital presence to showcase their portfolio of large-scale projects. The goal was to create a website that reflects their professionalism, expertise, and attention to detail.

The website serves as a portfolio hub for potential clients, featuring project galleries, service descriptions, and company credentials. It needed to convey trust, strength, and reliability — key values in the construction industry.

User Experience

I designed a user journey that guides visitors from the hero section through project showcases and finally to contact. The layout uses bold typography, industrial-inspired accents, and strategic whitespace to create a premium feel.

Smooth scrolling and subtle animations keep users engaged while maintaining professional restraint — nothing too flashy, just clean and effective.

The Problem

The client's previous website was outdated, slow, and difficult to navigate. Heavy imagery caused lag, especially on mobile devices. The site lacked proper structure for SEO and had no CMS for easy content updates.

Additionally, typical Webflow animations caused performance issues, making the site feel clunky and unprofessional.

The Solution

I rebuilt the site from scratch in Webflow, implementing Lenis for buttery-smooth scroll and GSAP for lightweight scroll-triggered animations. All images were optimized and lazy-loaded for performance.

The new structure follows SEO best practices, and the CMS allows the client to update project galleries and testimonials independently.

Responsive Implementation

The website was built with a mobile-first approach, ensuring a seamless experience across all devices from smartphones to 4K displays.

  • Fluid grid system for flexible layouts
  • Responsive typography with clamp()
  • Touch-optimized navigation menus
  • Adaptive image sizing per breakpoint
  • Consistent UX across all screen sizes

Performance Optimization

I implemented several techniques to ensure fast loading times and smooth interactions, even with heavy image content.

  • Lazy loading for all images
  • Minified CSS and JavaScript assets
  • Webflow native CDN optimization
  • Compressed and WebP images
  • 95+ Google PageSpeed score
