Alia USA - Organization Website

Project Overview

Alia USA is a professional organization that needed a complete digital presence. The goal was to create a website that reflects their mission, builds trust with their audience, and provides an easy way for the team to manage content without technical assistance.

The website serves as a central hub for information about their programs, events, and initiatives. It needed to be visually appealing, highly functional, and scalable for future growth.

User Experience

I focused on creating an intuitive navigation structure that makes it easy for visitors to find what they need. The design uses clean layouts, strategic whitespace, and clear calls-to-action to guide users through the content.

The result is a seamless experience that feels both professional and welcoming — encouraging users to explore and engage with the organization's mission.

The Challenge

The client had no coding experience and needed a website they could update independently. The site required complex multi-level navigation, dynamic content sections, and a blog/news system that would be easy to maintain.

Additionally, the design needed to work across all devices and load quickly despite having multiple large images and media assets.

My Solution

I designed the UI in Figma first, ensuring pixel-perfect precision before development. Then I built the site in Webflow using a modular CMS structure with reusable components. This allows the client to add/edit content easily without touching code.

I also implemented custom form submissions, dynamic filtering, and responsive image optimization for fast loading across all devices.

Responsive Design

The website was built with a mobile-first approach, ensuring a seamless experience across all screen sizes.

  • Fluid typography system
  • Flexible grid layouts
  • Touch-friendly navigation
  • Optimized images per device
  • Consistent UX across breakpoints

Performance Optimization

I implemented several performance techniques to ensure fast loading times and smooth interactions.

  • Lazy loading for images
  • Minified CSS/JS assets
  • Webflow native optimization
  • Compressed media files
  • 95+ PageSpeed score
